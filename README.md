## 🚀 About Me 
I'm a Data Analyst with around 2 years of experience turning diverse, messy data sources into clear, actionable insights that support real business decisions. My day-to-day involves pulling data from SQL Server and cloud platforms, building ETL pipelines, and designing Power BI dashboards and DAX-driven reports that help teams act quickly and confidently.

My path into analytics runs through public-sector data operations and community program coordination, which is where I picked up a strong discipline around data quality, security, and stakeholder communication — habits that now shape how I approach every dashboard and dataset.

I'm proficient in Excel (advanced formulas, Pivot Tables, Power Query, VBA/Macros), SQL (DDL, DML, DQL, DCL, window functions), and Power BI/Tableau for modelling and visualization, backed by a solid business-analysis toolkit — process re-engineering, stakeholder management, and Agile/Waterfall delivery. I hold an MSc in Accounting and Finance and a BSc in Agriculture. I love the moment data and process come together to reveal a clear path forward for a team.
